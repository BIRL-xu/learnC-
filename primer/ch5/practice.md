#### swich语句注意点：
* case标签必须是整型常量表达式:
````c++
char ch = getVal();
int ival = 42;
switch(ch){
case 3.14: // 错误：必须是整型
case ival: // 错误：必须是常量
}
````
* 变量必须定义在块内
````c++
ｃase true:
{
    string file_name;
    int ival = 0;
    int jval;
    break;
}
case false:
if(file_name.empty()) // 错误：　file_name不在作用域之内。
````
